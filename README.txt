# GVRP_IA
Resolución del Green Vehicle Routing Problem utilizando la técnica de Simulated Annealing

El output del programa, además de la información solicitada, también contiene la ruta de la solución inicial, de la cual comienza la exploración y explotación, junto a la distancia total de ella.

Para compilar el código, simplemente se realiza un "make", el que generará un ejecutable llamado "SA"

La forma de correr el programa es:

./SA instancia N

Donde:
- SA es el nombre del ejecutable generado con make
- instancia es un archivo txt con la información necesaria para utilizar el programa
- N es el número de iteraciones que se le dará al programa
